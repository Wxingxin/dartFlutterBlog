
## 🧩 一、什么是类（Class）

**类（Class）** 就是一种 **“模板”**，用来描述对象的 **属性（数据）** 和 **方法（行为）**。
比如：你要描述一个人，你可以写一个类：

```dart
class Person {
  String name = '未命名'; // 属性（字段）
  int age = 0;            // 属性（字段）

  void sayHello() {       // 方法
    print('你好，我是$name，今年$age岁');
  }
}
```

🧠 解释：

* `class` 关键字定义类。
* 类里面可以有：

  * **属性（Fields）**：用来<font color=red>存储数据</font>，比如 `name` 和 `age`。
  * **方法（Methods）**：定义<font color=red>行为</font>，比如 `sayHello()`。

---

## 🧍‍♂️ 二、什么是对象（Object）

**对象** 就是根据类创建出来的一个“实际的实例”。
比如我们有了 `Person` 这个模板，就可以创建很多“人”：

```dart
void main() {
  var p1 = Person(); // 创建对象
  p1.name = '小明';  // 给对象的属性赋值
  p1.age = 18;
  p1.sayHello();     // 调用对象的方法

  var p2 = Person(); // 再创建一个对象
  p2.name = '小红';
  p2.age = 20;
  p2.sayHello();
}
```

🧠 解释：

* `Person()` 就是在调用类的构造函数（即使我们还没学构造函数，也默认有一个）。
* 每个对象都有自己独立的数据。
* 你可以通过 `对象.属性` 和 `对象.方法()` 来访问。

输出结果：

```
你好，我是小明，今年18岁
你好，我是小红，今年20岁
```

---

## 🔍 三、类的属性和方法访问

1. 访问属性：

   ```dart
   print(p1.name);
   ```
2. 修改属性：

   ```dart
   p1.age = 25;
   ```
3. 调用方法：

   ```dart
   p1.sayHello();
   ```

---

## ⚙️ 四、类中也可以使用函数逻辑

你可以让类的方法中包含一些计算逻辑，比如：

```dart
class Circle {
  double radius = 1.0;

  double getArea() {
    return 3.14 * radius * radius;
  }

  void showInfo() {
    print('半径为 $radius 的圆，面积是 ${getArea()}');
  }
}

void main() {
  var c = Circle();
  c.radius = 5;
  c.showInfo();
}
```

输出：

```
半径为 5.0 的圆，面积是 78.5
```

🧠 说明：

* 方法可以返回值。
* 你可以在类中方法里调用其他方法。

## 📦 五、小结表格

| 概念         | 说明        | 示例                     |
| ---------- | --------- | ---------------------- |
| 类（Class）   | 定义一类对象的模板 | `class Person { ... }` |
| 对象（Object） | 类的实例      | `var p = Person();`    |
| 属性（Field）  | 用来存数据     | `String name = '小明';`  |
| 方法（Method） | 定义行为      | `void sayHello() {}`   |
| 访问属性       | 使用点号访问    | `p.name`               |
| 调用方法       | 使用点号调用    | `p.sayHello()`         |
